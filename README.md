# Real World Demo

This project demonstrates the Real World application features, including sign-in, POYSHA wallet functionality, market buy/sell operations, and a bilingual UI (English/Bangla). 

## Features
- Sign-in functionality
- POYSHA wallet integration
- Market buy/sell capabilities
- UI support for English and Bangla
- Implementation using Phaser 3

## Usage
To run the demo, open `index.html` in your web browser.
